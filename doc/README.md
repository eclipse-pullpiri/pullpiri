# Pullpiri Documentation

This directory contains all documentation for the Pullpiri project, organized by topic.

## Directory Structure

```
doc/
├── README.md           # This file
├── architecture/       # System architecture and design documents
├── assets/             # Images used in documentation
│   ├── diagram/        # Architecture and component diagrams
│   └── logo/           # Project logo files
└── guides/             # User and developer guides
```
