## about scenarios

### Deploy new scenario

```plaintext
POST /scenario
```

#### Parameters

None

#### Request body

{scenario_name}/{scenario_file_name}

Example

```text
bms/bms-high-performance
```

#### Response

| Code  | Description |
| ------| -----       |
| 200   | Success     |
| 404   | Fail        |

```json
# Success
{
    "resp" : "Ok"
}

# Fail
{
    "resp" : "Error message"
}
```

### Delete scenario

```plaintext
DELETE /scenario/{scenario_name}
```

#### Parameters

scenario name you want to delete

#### Request body

None

#### Response

| Code  | Description |
| ------| -----       |
| 200   | Success     |
| 404   | Fail        |

```json
# Success
{
    "resp" : "Ok"
}

# Fail
{
    "resp" : "Error message"
}
```
